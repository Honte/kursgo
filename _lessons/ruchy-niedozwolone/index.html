--- title: Ruchy niedozwolone ---

Kamienie możesz kłaść na planszy prawie wszędzie.
Istnieje jednak kilka miejsc, w których zasady zabraniają stawiać kamienie.

{% diagram illegal-1 %}

Czarny nie może wykonać ruchu w A. Biały może.

Dlatego, że gdy czarny tam zagra, jego kamień będzie całkowicie otoczony.

Takie samobójcze postępowanie jest zabronione.

{% enddiagram %} {% diagram illegal-2 %}

{% enddiagram %} {% diagram illegal-3 %}

Wraz z położeniem czarnego kamienia w B, dwa sąsiednie białe kamienie będą otoczone i usunięte.
