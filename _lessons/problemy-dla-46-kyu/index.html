Ruch czarnych. Na pierwszy rzut oka wygląda to na ruch zakazany...

{% endproblem %} {% problem 46K-2 %}

4 czarne kamienie są w niebezpieczeństwie!

Zabij białe kamienie, zanim sam zostaniesz zabity!

{% endproblem %} {% problem 46K-3-2shi %}

Zdobądź dwa białe kamienie przytulone do czarnych.

W poniższym przykładzie nie możesz tego zrobić w jednym ruchu.

{% endproblem %} {% problem 46K-4-kake %}

Jeśli potrafisz zdobyć biały kamień oznaczony literą A, który rozdziela czarne kamienie, ta gra stanie się dla ciebie łatwa.

[Podpowiedź] Nie próbuj atakować przeciwnika bezpośrednio.

{% endproblem %}

Dlaczego zabijamy kamienie przeciwnika?

Ponieważ przez usunięcie kamieni przeciwnika, które rozdzielają nasze kamienie, nasze rozłączone kamienie będą się mogły połączyć.

To jest bardzo ważny aspekt Go.

Zapamiętaj, że gdy tylko możesz, graj tak, by wszystkie Twoje kamienie były połączone.
