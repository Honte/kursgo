--- title: Defensywa ---

Teraz wiesz, że połączone ukośnie kamienie mogą być rozcięte przez przeciwnika. Zanim zostaną rozcięte możesz się jednak ubezpieczyć.

{% diagram 0 %}

Jeżeli biały zagra tutaj cięcie w F6, dwa czarne kamienie znajdą się w niebezpieczeństwie.

Dlatego czarny powinien zagrać ruch ubezpieczający przed cięciem.

Jest to ruch łączący.

{% enddiagram %}

{% diagram 1 %}

Tutaj, po F6, czarny mocno łączy swoje kamienie.

Jest to twarde łączenie. Biały nie bedzie miał już możliwości ich przecięcia.

{% enddiagram %}

Poniżej możesz zobaczyć luźne łączenie.

{% freeplay 2 %}

Biały może oczywiście zagrać F6, jednak zostałby od razu zabity ruchem w F5. Spróbuj proszę grając na planszy przeanalizować to samemu.

Kamienie są zatem wirtualnie połączone.

{% endfreeplay %}

Luźny rodzaj połączenia jest nieco bardziej rozciągnięty niż połączenie twarde.

Jeżeli wiesz kiedy zagrać we właściwym miejscu luźne łączenie, nie będziesz już długo początkującym.

{% freeplay 3 %}

Na tym diagramie kamienie czarnego mają dwa miejsca cięcia - w E5 i F4. Wydaje się, że któryś z nich będzie przecięty...

Istnieje jednak ruch, który chroni obydwa miejsca cięcia jednocześnie!

Zobacz proszę ruch w F5.

Teraz białe nie mogą grać ani E5, ani F4.
Stało się tak, ponieważ F5 łączy luźnym połączeniem oba miejsca jednocześnie.

{% endfreeplay %}
