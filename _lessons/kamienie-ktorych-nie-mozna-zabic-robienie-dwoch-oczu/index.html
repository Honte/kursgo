Kamienie, których nie można zabić

Jak zapewne spostrzegłeś w ostatnim przykładzie, nawet grupa składająca się z wielu kamieni może zostać zabita, gdy zostanie otoczona.

{% diagram 0-can-kill %} Na przykład, na tym diagramie czarne kamienie zostały dokładnie otoczone - nie mają dokąd uciec - można tylko czekać na zabicie przez białe ruchem w środek. {% enddiagram %}

A co w sytuacji poniżej?

{% diagram 1-two-eyes %} Pomimo że czarne są otoczone, białe nie mogą ich zabić. Są dwa miejsca potrzebne do zabicia czarnej grupy - ale oba otoczone przez czarne, więc biały nie może w nie grać. {% enddiagram %}

Dlatego gdy masz 2 odseparowane miejsca, czyli dwoje oczu, Twoje kamienie nigdy nie będą zabite.
O takich grupach kamieni mówimy, że są żywe.

Czarne są znowu zadowolone.
