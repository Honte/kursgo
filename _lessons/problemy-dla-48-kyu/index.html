--- title: Problemy dla 48 kyu ---

Zdobywanie kamieni różnymi metodami!

Od teraz nie zawsze będziesz mógł osiągnąć sukces w jednym ruchu.

{% problem 48K-1-3shi %}

Zdobądź trzy białe kamienie!

Możesz to zrobić w dwóch ruchach, jednak decydujące znaczenie ma pierwszy ruch.

{% endproblem %}

{% problem 48K-2-hashi %}

Weź do niewoli dwa białe kamienie z lewej krawędzi.

Ciężko jest dokądkolwiek uciec kamieniami stojącymi na krawędzi.

{% endproblem %}

{% problem 48K-3-many %}

Za dużo kamieni! Bądź ostrożny z kamieniami w Atari.

Jeśli popełnisz błąd, zostaną zabite Twoje kamienie!

{% endproblem %}
