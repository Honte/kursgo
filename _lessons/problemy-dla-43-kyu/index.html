--- title: Problemy dla 43 kyu ---

Semeai (wzajemny atak)

{% problem 43K-1-semeai %}

Ruch czarnych. Dwie grupy wzajemnie się otaczają.
Albo 3 czarne kamienie, albo 3 białe zginą.

{% endproblem %}

W sytuacji gdy czarne i białe otaczają się wzajemnie i żadna z wewnętrznych grup nie ma dwojga oczu, obie strony próbują wzajemnie wypełnić puste miejsca zwane oddechami, aby zabić wewnętrzne kamienie przeciwnika.
Taka sytuacja jest zwana "Semeai" - wzajemny atak.

W powyższym problemie, obie strony mają po 3 oddechy. Jednak czarny może zabić białe, gdyż jest szybszy, ponieważ jest ruch czarnych.
Gdyby ruch należał do białych, to białe wygrałyby Semeai.

{% problem 43K-2-semeai %}

Czarny może wygrać Semeai.

{% endproblem %}

{% problem 43K-3-semeai %}

Uratuj 3 czarne kamienie stojące samotnie w rogu!

{% endproblem %}
