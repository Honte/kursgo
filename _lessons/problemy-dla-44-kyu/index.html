Robienie dwojga oczu

Teraz czarny jest całkowicie otoczony. Jaki musi zrobic ruch?

{% problem nigan-iki-1 %} {% endproblem %}

Pomimo że czarne mają w środku trzy pola, ta figura może mieć wkrótce tylko jedno oko.
Dlatego musisz zrobić dwoje oczu przez postawienie kamienia rozdzielającego.

Jeżeli tego nie zrobisz biały zagra tam, gdzie powinien zagrać czarny.

Przysłowie Go - Punkt decydujący dla Ciebie jest punktem decydującym dla przeciwnika.

Gdzie powinieneś postawić kamień rozdzielający na diagramie poniżej?

{% problem nigan-iki-2 %} {% endproblem %}

Jak widać Twoje terytorium jest zbyt małe, czeka Cię ciężkie życie.

Zawsze próbuj zbudować wystarczająco duże terytorium i nie dać się otoczyć.

A co z przykładem poniżej?

W tym przykładzie nie musisz kłaść kamienia rozdzielającego.

Gdy biały zrobi ruch w a, odpowiesz grając w b.
Gdy biały zagra w b, odpowiesz w a.

Możesz więc zawsze zrobić dwoje oczu, gdy biały zaatakuje.

Jednak gdybyś zignorował atak białych, biały zajmie punkty a i b równocześnie.
W takim przypadku będziesz miał tylko jedno oko i zginiesz.
